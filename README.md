Attendance Tracker

Project Description
A Python-based command-line attendance tracker that records student names, validates entries, displays formatted attendance summaries, and optionally saves reports to a text file.

Files Included
- `tracker.py` — main program file
- `sample_run_1.txt` — example run 1
- `sample_run_2.txt` — example run 2
- `sample_run_3.txt` — example run 3
- `attendance_log.txt` — generated attendance report
